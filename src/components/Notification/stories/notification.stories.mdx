悬浮出现在页面角落，显示全局的被动通知提醒消息。

###  使用场景及规则
* 较为复杂的通知内容;
* 显示一段时间（默认为 3 秒，可自定义时长）后自动消失；
* 带有交互的通知，给出用户下一步的行动点；
* 不随页面滚动，可以让 Notification 从屏幕四角中的任意一角弹出。

### 基础样式
通知弹出后将在 3s 后自动关闭，可手动点击关闭按钮关闭提示。
<Preview>
  <Button onClick={() => {Notification.open({title: '标题', type: 'info'})}}>
    仅通知标题
  </Button>
  <Button onClick={() => {Notification.open({title: '标题', description: '这是描述这是描述这是描述这是描述这是描述', type: 'info'})}}>
    通知标题+描述
  </Button>
   <Button onClick={() => {Notification.open({title: '自定义图标', description: '这是自定义图标描述', type: 'info', icon: 'search'})}}>
    自定义图标
  </Button>
  <Button onClick={() => {Notification.open({title: '不带图标', description: '这是不带图标的描述'})}}>
    不带图标
  </Button>
  <Button onClick={() => {Notification.open({title: '不带关闭按钮', description: '这是不带关闭按钮的描述', type: 'info', closable: false})}}>

  </Button>

带有 icon，常用来显示「成功、警告、消息、错误」类的系统消息
