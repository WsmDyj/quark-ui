用于展示 ICON。该组件的 ICON 图形基于 [Font Awesome](https://fontawesome.com/icons?d=listing&m=free)中提供的免费开源版本库

### 使用指南
使用 **Icon** 组件，指定图标对应的 className 属性，示例代码：
```tsx

<Icon className="address-book" />

```

### 其他属性
使用spin该类可旋转任何图标，pulse类用八个步骤来旋转，rotation旋转角度，size图标大小
<Preview>

  <Icon className="spinner" size='2x' pulse />
  <Icon className="ad" size='2x' rotation = {90} />
</Preview>
